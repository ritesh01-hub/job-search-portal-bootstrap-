# 💼 Job Search Portal

A responsive **Job Search Portal** web application built with **HTML**, **CSS**, and **Bootstrap 4**. This project allows users to search for jobs by keywords, location, and company, and browse the latest job listings.

## ✨ Features

- 🔍 **Job Search** — Search by keywords, location, and company name
- 📋 **Job Listings** — Browse latest job cards with company, location, and description
- 📱 **Responsive Design** — Mobile-friendly layout using Bootstrap 4 grid system
- 🎨 **Clean UI** — Simple, readable interface with Bootstrap components
- 🧭 **Navigation Bar** — Links for Home, Jobs, Companies, About Us, and Contact

## ⚙️ Getting Started

### Prerequisites

No installations required! Just a modern web browser.

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/<your-username>/job-search-portal.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd job-search-portal
   ```

3. **Open `index.html` in your browser**
   ```bash
   # On Windows
   start index.html

   # On macOS
   open index.html

   # On Linux
   xdg-open index.html
   ```
